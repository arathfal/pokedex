# Pokedex

## Installation

1. Clone the repository:

```bash
git clone https://github.com/arathfal/pokedex.git
```

2. Navigate to the root directory:

```bash
cd pokedex
```

3. Install dependencies :

```bash
yarn install
```

## Usage

To run the server in development mode:

```bash
yarn dev
```

To build the code:

```bash
yarn build
```

To start the server in production mode:

```bash
yarn start
```
